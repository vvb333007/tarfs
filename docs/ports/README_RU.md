## Поддержка STM32

Установите файл `os_stm32.c` вместо `os_esp32.c` (os_esp32.c удалите, а os_stm32.c скопируйте).

Настройте макросы `TARFS_ADDRESS` и `TARFS_SIZE` в соответствии с вашей 
целевой платформой (см. комментарии в `os_stm32.c`).

Код можно и нужно использовать как основу для реализации платформенного слоя TARFS для вашего проекта 
на базе STM32.
